# Offline Notes PWA

A simple Progressive Web App (PWA) for taking notes offline. Write and save notes directly in your browser, even without an internet connection.

## Features

- Create, save, and delete notes easily  
- Works offline using Service Worker caching  
- Installable on your device's home screen as a standalone app  
- Responsive and mobile-friendly design  

## How to Use

1. Open the app in your browser.  
2. Write your note in the textarea.  
3. Click **Save Note** to add it to your list.  
4. View saved notes below, and delete any note by clicking the red **Delete** button.  
5. Install the app by clicking the **Install App** button when it appears.

## Installation

- This app supports installation as a PWA on supported browsers (Chrome, Edge, Firefox Android).  
- Click the **Install App** button or use your browser's install option to add it to your device.

## Technology Stack

- HTML, CSS, and JavaScript  
- Service Worker for offline functionality  
- Web Storage API (localStorage) for saving notes  

## Development

- Clone the repository  
- Serve files via any static server (e.g., `live-server` or GitHub Pages)  
- Access the app at `http://localhost` or your hosting URL
